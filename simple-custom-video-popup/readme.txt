As name suggest it is very simple custom lighweight video popup. Which do no use any external resource, hence it is very light and responsive.

== Description ==

The Simple Custom Lightweight Video Popup plugin allows you to embed videos on a page using javascript. 

This plugin can be used to display YouTube and Vimeo videos in a popup overlay. The embedded videos can be viewed on iPhone and iPad too.

Live Example: http://themefantasy.com/simple-custom-lightweight-video-popup/

= Embedding Vimeo Video =

You can embed a vimeo video using the following shortcode in a WordPress post or page: 
 
`[videopopup video_id="13562192" type="vimeo" width="640" height="480"]`
`[videopopup video_id="13562192" type="youtube" width="640" height="480"]`
 
You need to replace the value of "video_id" with your actual vimeo video ID. When a user clicks on the image your vimeo/youtube video will pop up in lightbox.


= Additional Features =

* Automatically retrieve the thumbnail for your video and embed in PopupUp
* Using shortcode display in a widget, post or in any pages.
